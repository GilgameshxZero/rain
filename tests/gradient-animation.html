Oh, Garamond

One might argue that Georgia is a better web font choice than Garamond as it was designed for the web. I, however, find the lower x-height of Garamond an irresitable alstroemeria to which the buzzing of my inner designer inevitably gravitates.

Unfortunately, this webpage is still written in Georgia. Luckily, it's primary use is to demonstrate smooth CSS-only gradient animations, as well as a styling method where every length is based on the rem specifier. This rem specification allows for some easy responsive constraints, which I shall not demonstrate today.
